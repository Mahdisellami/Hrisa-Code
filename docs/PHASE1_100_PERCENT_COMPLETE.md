## Lessons Learned

### What Worked Exceptionally Well
1. **Static Analysis First**: Eliminating LLM from discovery was key
2. **Content Validation**: Catching bad output early prevented issues
3. **Directive Prompts**: Clear, specific instructions produced better results
4. **Hybrid Approach**: Combining static + template + LLM was optimal

### What We'd Do Differently
1. **Start with static analysis**: Should have done this from day one
2. **Validate earlier**: Content validation should be in initial design
3. **Template more**: LLM should only write prose, not structure

### Key Insights
- LLMs are great at prose, terrible at discovery
- Static code analysis is instant and perfect
- Validation is not optional - it's essential
- Templates ensure consistency better than LLM
